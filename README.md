# Cognizant Java Full Stack Engineering (Java FSE) - Training Repository

## 📚 Overview

This repository contains hands-on practice exercises and projects from the **Cognizant Digital Nurture Program** for **Java Full Stack Engineering (Java FSE)**. This is a comprehensive training initiative designed to upskill developers in building enterprise-grade full-stack applications using Java technologies.

---

## 🎯 Training Objectives

The Cognizant Java FSE training program aims to equip developers with:

- **Core Java Concepts**: Object-oriented programming, design patterns, and best practices
- **Web Development**: Building scalable web applications with Spring Framework
- **Database Management**: Working with relational and NoSQL databases
- **Frontend Development**: Modern UI frameworks and responsive design
- **Cloud & DevOps**: Containerization, CI/CD pipelines, and cloud deployment
- **Microservices Architecture**: Building distributed systems and service-oriented architecture
- **Testing & Quality Assurance**: Unit testing, integration testing, and test automation

---

## 📋 Mandatory Hands-On Exercises

The training includes mandatory hands-on practice exercises covering the following modules:

### **Module 1: Java Fundamentals**
- Data Types and Variables
- Control Flow Statements
- Object-Oriented Programming (OOP) Concepts
- Exception Handling
- Collections Framework
- File I/O Operations

### **Module 2: Advanced Java**
- Multithreading and Concurrency
- Stream API and Functional Programming
- Reflection and Annotations
- Lambda Expressions
- Regular Expressions

### **Module 3: Database Technologies**
- SQL Fundamentals (DDL, DML, DQL)
- JDBC and Database Connectivity
- Hibernate ORM Framework
- JPA (Java Persistence API)
- Database Design and Optimization
- NoSQL (MongoDB/Firebase)

### **Module 4: Spring Framework**
- Spring Core (IoC, Dependency Injection)
- Spring MVC (Model-View-Controller)
- Spring Boot (Microservices, Auto-configuration)
- Spring Data (Data Access Layer)
- Spring Security (Authentication & Authorization)
- RESTful Web Services (REST APIs)

### **Module 5: Frontend Technologies**
- HTML5 and CSS3
- JavaScript (ES6+)
- Angular / React Framework
- Responsive Design
- Frontend Integration with Backend APIs

### **Module 6: Microservices & Cloud**
- Microservices Architecture Patterns
- Docker Containerization
- Kubernetes Orchestration
- Cloud Platforms (AWS/Azure/GCP)
- CI/CD Pipelines (Jenkins, GitLab CI)
- API Gateway and Service Discovery

### **Module 7: Testing & DevOps**
- Unit Testing (JUnit, Mockito)
- Integration Testing
- Test-Driven Development (TDD)
- Performance Testing
- Continuous Integration & Continuous Deployment
- Infrastructure as Code (IaC)

### **Module 8: Project Implementation**
- End-to-End Project Development
- Real-world Scenario Implementation
- Agile Methodology Application
- Code Quality and Best Practices
